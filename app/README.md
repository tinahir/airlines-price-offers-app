# Airlines price offers

# Getting Started

### Prerequisites

You'll need [Node](https://nodejs.org/en/), [Yarn](https://yarnpkg.com/en/).

## Run the app

```sh
yarn install
yarn start
```

Runs the app in the development mode.<br />
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

## Build the app

```sh
yarn build
```

## Run unit test

```sh
yarn test
```
